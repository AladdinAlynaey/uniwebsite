API Key Configuration

Add API keys to your .env file:

Get your API key from Google AI Studio

# Add multiple keys for fallback
GEMINI_API_KEY_1=your_first_key
GEMINI_API_KEY_2=your_second_key
GEMINI_API_KEY_3=your_third_key
GEMINI_MODEL_NAME=gemini-2.0-flash

Get your API key from OpenRouter (includes free models!)

OPENROUTER_API_KEY=your_openrouter_key
OPENROUTER_MODEL=google/gemini-2.0-flash-exp:free

Free models include: google/gemini-2.0-flash-exp:free, meta-llama/llama-3.2-3b-instruct:free, qwen/qwen-2-7b-instruct:free

Get your API key from Groq Console (free tier available!)

GROQ_API_KEY=your_groq_key
GROQ_MODEL=llama-3.3-70b-versatile

Available models: llama-3.3-70b-versatile, llama-3.1-70b-versatile, mixtral-8x7b-32768

Telegram Bot Settings
@
This is used to create links to your bot.

To create a new Telegram bot and get a token:

  1. Open Telegram and search for @BotFather
  2. Send the command /newbot and follow the instructions
  3. Copy the token provided by BotFather and paste it here
  4. Copy the username of your bot (without @) and paste it above
Webhook Setup Instructions

After setting your token, you need to set up a webhook for your bot. Use the following URL format:

https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=YOUR_SERVER_URL/api/telegram/webhook

Replace YOUR_TOKEN with your bot token and YOUR_SERVER_URL with your server's URL.
